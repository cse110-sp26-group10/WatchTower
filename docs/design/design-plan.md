# Watchtower Design

Figma used to create the WatchTower designs/wireframes, which can be found [here]. (https://www.figma.com/design/rFF9NQnwunN9XZw9jF6BNm/WatchTower-Product---Design?node-id=102-2&t=OmeIaFYzFefQiq02-1). Keep in mind that this is a living document that may change as the project goes.

## Wireframe Diagram

Low fidelity wireframes were created for 4 important pages: The dashboard, the issue detail, the user feedback page, and the deployments page. Arrows show how each page is connected. The wireframe includes all the features that we expect to have in our MVP.
